Τhe attribute <title>

The <title> attribute is not an acceptable substitute for an <alt> attribute: try to avoid duplicating the alt attribute's value in a title attribute declared on the same image. Doing so may cause some screen readers to announce the description twice, creating a confusing experience. <title> 's value is to be used as a tooltip, which appears shortly after the cursor stops moving over the image. While this can provide additional information to the user, you should not assume that the user will ever see it. If you have information that's particularly important or valuable for the user to see, you should present it using <figure> or <figcaption> instead of using <title>.
