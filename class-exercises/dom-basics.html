Basic DOM Properties & Methods


PROPERTY/METHOD ACTION CODE
document.querySelector( CSS Selector ) Get HTMLElement let button = document.querySelector( "button" );
document.querySelectorAll( CSS Selector ) Get List of HTMLElement let buttons = document.querySelectorAll( "button" );
ELEMENT.innerHTML Get HTML Content of HTMLElement let content = button.innerHTML;
ELEMENT.innerHTML = "<p>Hello<p>" Set HTML Content of HTMLElement let content = button.innerHTML = "Buy <strong>now</strong>";
ELEMENT.getAttribute( NAME ) Get HTMLElement Attribute button.getAttribute( "class" );
ELEMENT.setAttribute( NAME, VALUE ) Set HTMLElement Attribute button.setAttribute( "class", "dark" );
document.createElement( TAG ) Create new HTMLElement let button = document.createElement( "button" );
NODE.appendChild( NODE ) Append a NODE at the end of another NODE document.body.appendChild( button );
  • CSS Selector: e.g. #someId, .someClass, h1, p, section .child, etc.
  • TAG: e.g. "h1", "p", "div", "section", etc.
  • HTML Content: HTML in the form of a String, e.g. <div>Hello World</div>
  • HTMLElement: An HTML Element DOM Node, e.g. h1, div, body, etc.
  • DOM Node: Any kind of DOM Node, e.g.: HTMLElement, Text Content, Attribute, Comment, etc.
